# 发布说明

## v1.0.0 (2025-11-30)

### 新功能

- **单文件解密** - 支持选择单个 `.txt` 文件进行解密
- **批量解密** - 支持选择目录批量解密所有 `.txt` 文件
- **多线程处理** - 4 线程并发处理，提升批量解密速度
- **编码自动识别** - 支持 UTF-8、UTF-16 LE、UTF-16 BE 编码
- **实时进度显示** - 显示解密进度条和详细日志
- **详细统计信息** - 批量解密完成后显示成功/失败统计
- **Material Design 3** - 现代化的 UI 设计
- **深色模式** - 跟随系统自动切换
- **流畅动画** - 精心设计的卡片动画效果

### 技术特性

- **Native 密钥保护** - 使用 C++ JNI 保护解密密钥
- **DocumentFile API** - 完美支持 Android 11+ 分区存储
- **SAF 框架** - 自动管理存储权限
- **内存优化** - 高效处理大文件
- **异常处理** - 完善的错误处理机制

### 系统支持

- **最低版本**: Android 5.0 (API 21)
- **推荐版本**: Android 11+ (API 30+)
- **完美支持**: Android 15 (API 35)
