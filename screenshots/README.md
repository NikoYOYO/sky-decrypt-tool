# 截图文件夹

这个文件夹用于存放应用截图。

## 需要的截图

请添加以下截图（推荐尺寸：1080x2340 或等比例）：

### 1. single_file.png
- **内容**：单文件解密界面
- **状态**：选择文件后，显示解密日志

### 2. batch_decrypt.png
- **内容**：批量解密界面
- **状态**：显示批量解密完成的统计信息

### 3. about_page.png
- **内容**：关于页面
- **状态**：显示完整的关于页面，包括开发者信息

## 拍摄建议

1. **使用浅色/深色模式**对比截图
2. **确保内容清晰**，文字可读
3. **隐藏敏感信息**（如果有）
4. **保持一致性**，使用相同设备截图

## 处理工具

- Android Studio Device Manager
- 真机截图 (音量下 + 电源键)
- adb 命令：`adb exec-out screencap -p > screenshot.png`
